## Setup

Install dependencies

```sh
$ npm install
```

## Development

```sh
$ npm start
# visit http://localhost:8080
```

- HTML 수정: `index.html`를 수정하시면 됩니다.
- JS 수정: `/app/index.js`를 수정하시면 됩니다.
- CSS 수정: `/assets/styles/index.css`를 수정하시면 됩니다.
